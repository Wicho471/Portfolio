Project Overview

RemoteHealth is a Java 17 and JavaFX application developed by Axolotlj Tech for remote monitoring and processing of biomedical signals from medical devices. It provides real-time analysis, modern visualization, and communication via WebSocket and serial interfaces. The system supports ECG, SpO₂, blood pressure estimation, temperature monitoring, and motion artifact rejection, following international medical standards. Data can be exported in CSV/JSON, visualized interactively, and integrated with Arduino/ESP32-based embedded systems for sensor acquisition (ECG, SpO₂, temperature, accelerometer). The embedded side manages concurrency with FreeRTOS, asynchronous communication (HTTP/WebSocket), and persistent configuration. RemoteHealth combines digital signal processing, anomaly detection, and advanced communication to deliver a reliable platform for biomedical data monitoring.

Key Features

Serial and WebSocket communication

Enables seamless data transmission from medical devices either via USB/serial ports or through real-time WebSocket connections over a network.

Digital Signal Processing (DSP)

Processes raw biomedical signals (ECG, SpO₂, PTT, temperature) using filtering, peak detection, and spectral analysis techniques to extract clinically relevant metrics.

Biomedical data export and analysis (CSV/JSON)

Allows saving and sharing collected data in common formats for further analysis or integration with external tools.

Visualization with JavaFX

Provides a modern and interactive graphical interface to display real-time biomedical signals, trends, and analysis results.

Anomaly detection in biomedical data

Identifies irregularities such as abnormal heart rate, oxygen saturation drops, or motion artifacts, helping in early detection of potential health issues.

Cross-platform availability

Distributed as executables and packages for all major platforms, including Windows (.exe), Linux (.deb), macOS (.dmg), Android (.apk), and iOS (.ipa), ensuring accessibility across desktop and mobile devices.
