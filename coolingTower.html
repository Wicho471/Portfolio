Project Information

  • Category: Embedded & GUI
  • Client: Chemistry Department of Technological Institute of Pachuca
  • Project date: November 25, 2024

Project Overview

This project involved the complete overhaul and modernization of a three-stage cooling tower for an academic laboratory, addressing the challenges of obsolete equipment and the need for safer, more efficient operations. A custom, low-cost monitoring and control system was engineered to transform the existing hardware into an effective, sustainable, and hands-on educational tool. The core of the solution is a robust integration between hardware and software, designed to be both functional and accessible for an academic environment.

The system's hardware layer is managed by an

Arduino microcontroller, which serves as the central hub for data acquisition from a suite of integrated sensors. These sensors capture critical process variables in real-time, including temperature (DS18B20), water flow (ARD-370), differential pressure (MPX5500DP), humidity (DHT11), and water levels, ensuring comprehensive oversight of the tower's performance. For the software layer, a dedicated desktop application was developed from the ground up using

Java, featuring an intuitive Graphical User Interface (GUI). This application provides dynamic visualization of all incoming data, allowing operators to monitor the system's status and control its functions effectively. Reliable, bidirectional serial communication was established between the Arduino and the Java application to ensure precise data transmission and control.

The final result is a fully operational prototype that not only restores the cooling tower's functionality but also enhances it with modern automation capabilities, creating a valuable platform for students to gain practical experience in process control, sensor integration, and software development. This project stands as an affordable and scalable alternative to costly industrial systems, with a design that promotes sustainability by optimizing energy and water consumption.

Key Features

Real-Time Data Monitoring

Implemented a system for live monitoring of critical variables including temperature, water flow, pressure, humidity, and water levels.

Custom Java GUI

Developed a user-friendly graphical interface from the ground up in Java, providing dynamic visualization of sensor data and system status for intuitive process control.

Microcontroller and Sensor Integration

Successfully integrated an Arduino microcontroller with a variety of sensors (DS18B20, ARD-370, MPX5500DP, DHT11) to create a cohesive data acquisition unit

Bidirectional Serial Communication

Established a reliable serial communication protocol between the Arduino and the Java application, allowing for both data transmission and remote control commands

Educational and Sustainable Design

The system was designed as an accessible educational platform for engineering students while optimizing the use of water and energy, promoting sustainability by retrofitting existing hardware

Low-Cost & Scalable Architecture

Created a functional and affordable alternative to expensive industrial SCADA systems, with a modular design that allows for easy replication and scalability in other academic laboratories
